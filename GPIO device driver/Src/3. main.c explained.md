### 🔹 1. GPIO_Init(GPIOA, &config)

**Step A: What is passed?**  
GPIOA → 0x40020000 (base address)  
&config → address of struct  

**Step B: Inside function**

```c
uint8_t pin = pinConfig->pin;
```

👉 pin = 5  

**Step C: MODER**

```c
port->MODER &= ~(0x3 << (2 * pin));
```

👉 (2 * 5) = 10  

```c
MODER &= ~(0x3 << 10);
```

👉 clears bits [11:10]  

```c
port->MODER |= (pinConfig->mode << (2 * pin));
```

👉 mode = 1 (OUTPUT)  

```c
MODER |= (1 << 10);
```

👉 sets bits to 01  

🔥 Result:  

Pin 5 mode = OUTPUT

### 🔹 2. GPIO_SetPin(GPIOA, 5)

**Inside:**

```c
port->ODR |= (1 << 5);
```

👉 sets bit 5 in ODR  

**Binary idea:**

Before:

```
00000000
```

After:

```
00100000
```

👉 Pin goes HIGH  

---

### 🔹 3. GPIO_TogglePin(GPIOA, 5)

**Inside:**

```c
port->ODR ^= (1 << 5);
```

👉 XOR flips bit  

**Case:**

If:

```
00100000
```

After toggle:

```
00000000
```

👉 HIGH → LOW  

---

### 🔹 4. GPIO_ReadPin(GPIOA, 5)

**Inside:**

```c
return (port->IDR >> 5) & 0x1;
```

**Step-by-step:**

IDR =

```
00100000
```

Shift:

```
00000001
```

Mask:

```
00000001
```

👉 Returns 1
